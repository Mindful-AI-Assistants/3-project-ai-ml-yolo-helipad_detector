# packages.txt
# System-level (Linux) dependencies installed via `apt-get` on Streamlit
# Community Cloud. This file is optional and is ONLY used when certain
# Python packages require native libraries that are not present by default.
#
# Location:
# - Recommended: repository root.
# - Each line must contain a single Debian package name (no versions).
#
# Purpose in this project:
# - OpenCV (even headless) needs libGL to load and process images in some
#   environments. Installing `libgl1` provides the `libGL.so.1` shared library
#   and fixes the "ImportError: libGL.so.1: cannot open shared object file"
#   error seen in Cloud logs for Ultralytics + OpenCV.

libgl1   # Provides libGL.so.1 needed by OpenCV; safe and commonly used on Streamlit Cloud.
